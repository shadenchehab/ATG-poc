# Alliance Technical Group — PoC Plan

Static site for the Alliance Technical Group Computer for Sales proof-of-concept plan.

## Deploy on Vercel

1. Import this repository at [vercel.com/new](https://vercel.com/new).
2. Select **ATG-poc** and deploy with default settings (no build step required).
3. Vercel serves `index.html` at the project root.

Or deploy from the CLI:

```bash
npm i -g vercel
vercel
```

## Local preview

Open `index.html` in a browser, or run:

```bash
npx serve .
```
